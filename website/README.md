# Security-Plus Website

This is the website for the Security-Plus project, built using modern web technologies to provide a user-friendly interface for letting people know about the project.

## ⚙️ Technologies Used

- Nextjs
- TypeScript
- TailwindCSS
- shadcn

## ⭐ Getting Started

1. **Installation**
   ```bash
   npm install
   ```

2. **Development**
   ```bash
   npm run dev
   ```

3. **Build**
   ```bash
   npm run build
   ```

4. **Testing**
   ```bash
   npm test
   ```
